# Partner Targets

## Integration Roadmap

### Phase 1: Core Integrations (Months 1-3)
1. **Stripe API**
   - **Rationale**: Stripe is a leading payment processor with a robust API and extensive documentation. It supports cross-border payments and has a large user base.
   - **Free-tier limits**: $50,000 in transactions per month.
   - **Integration effort**: Medium (M)
   - **Value-add**: Solves the need for seamless payment processing and fraud detection for banks and payment providers.

2. **PayPal API**
   - **Rationale**: PayPal is widely used for cross-border transactions and has a strong affiliate program.
   - **Free-tier limits**: $10,000 in transactions per month.
   - **Integration effort**: Medium (M)
   - **Value-add**: Provides a familiar payment gateway for users and leverages PayPal's affiliate program for revenue sharing.

3. **Blockchain.info API**
   - **Rationale**: Blockchain.info offers blockchain data and transaction services, which are essential for transparency and security in cross-border payments.
   - **Free-tier limits**: Limited to basic API calls.
   - **Integration effort**: Large (L)
   - **Value-add**: Enhances the transparency and security features of the platform, attracting users who prioritize these aspects.

### Phase 2: Strategic Integrations (Months 4-6)
4. **KYC/AML Solutions (e.g., Jumio, Onfido)**
   - **Rationale**: KYC/AML solutions are critical for regulatory compliance and user trust.
   - **Free-tier limits**: Varies by provider, typically limited to a certain number of verifications.
   - **Integration effort**: Large (L)
   - **Value-add**: Solves the need for robust identity verification and compliance, which is a significant pain point for banks and corporations.

5. **Digital Asset Exchange APIs (e.g., Binance, Coinbase)**
   - **Rationale**: Integrating with digital asset exchanges can attract users who want to use cryptocurrencies for cross-border payments.
   - **Free-tier limits**: Limited to basic API calls.
   - **Integration effort**: Large (L)
   - **Value-add**: Provides a seamless experience for users who want to use cryptocurrencies, enhancing the platform's appeal to digital asset exchanges.

### Phase 3: Niche Integrations (Months 7-9)
6. **Fraud Detection Services (e.g., Sift, Signifyd)**
   - **Rationale**: Fraud detection services can help reduce the risk of fraudulent transactions.
   - **Free-tier limits**: Varies by provider, typically limited to a certain number of transactions.
   - **Integration effort**: Medium (M)
   - **Value-add**: Solves the need for advanced fraud detection, which is a significant pain point for payment providers.

7. **Local Payment Providers (e.g., Alipay, WeChat Pay)**
   - **Rationale**: Integrating with local payment providers can attract users in specific regions and enhance the platform's global reach.
   - **Free-tier limits**: Varies by provider, typically limited to a certain number of transactions.
   - **Integration effort**: Large (L)
   - **Value-add**: Provides a seamless payment experience for users in specific regions, enhancing the platform's appeal to banks and payment providers.

## Affiliate/Revenue-Share Opportunities

- **Stripe**: Affiliate program with revenue sharing.
- **PayPal**: Affiliate program with revenue sharing.
- **Binance**: Affiliate program with revenue sharing.
- **Coinbase**: Affiliate program with revenue sharing.

## Integration Effort Legend

- **Small (S)**: Minimal effort, well-documented APIs.
- **Medium (M)**: Moderate effort, requires some custom development.
- **Large (L)**: Significant effort, complex APIs or custom development required.
